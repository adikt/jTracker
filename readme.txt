Job Search Tracker
web based tool for helping keep track of the job application process by organizing company pros, cons, research, contacts, application steps taken and suggested next steps.
Far from complete or refined but free to use.

Utilizes the following libraries:

uMScript v.1.0.5 By Evans Kirimi
Scriptaculous Help Baloon Sript By Beau D. Scott
jQuery JavaScript Library v1.10.2 from jQuery Foundation, Inc.
Colorbox v1.5.13 By Jack Moore
